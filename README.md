# Social Media Analytics Dashboard (Power BI)

## Overview
This project analyzes user engagement patterns across a social media platform using Power BI. The dashboard provides actionable insights into user behavior, demographics, and content performance.

## Key Insights
- Engagement KPIs: Likes (46K), Shares (22K), Comments (11K)
- Video performance analysis (3-sec vs 1-min views)
- Hobby-based engagement trends
- Profession-wise video consumption patterns
- Gender and age demographic distribution

## Features
- Interactive slicers for Gender, Profession, Country, and Hobby
- KPI cards for engagement tracking
- Visual storytelling through bar charts, donut charts, and histograms

## Tech Stack
Power BI, DAX, SQL, Python (data preprocessing)

## Business Impact
Helps marketing teams identify:
- high-engagement audience segments
- content strategy improvements
- demographic targeting opportunities
